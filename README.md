# VENV setup
## Creating a venv
Run the command

```
python -m venv projvenv
```
## Activating venv
In a bash shell

```
source projvenv/Scripts/activate
```

# Installing Dependencies
Once venv is activated, run:
```
pip install -r requirements.txt
```
to install all necessary packages/modules.
